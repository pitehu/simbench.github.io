Data Format

All datasets have been standardized into a unified format with the following fields:

dataset_name

Name of the original dataset

group_prompt_template

Persona/grouping instruction template

group_prompt_variable_map

Variables to fill into the persona template

input_template

The question text

human_answer

Dictionary of option labels and human response proportions

group_size

Number of human respondents
